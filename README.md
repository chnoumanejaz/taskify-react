## Project Overview
Taskify represents a meticulous blend of design and development expertise, providing a comprehensive solution for seamless project workflow management.

## Quick Demo
[Taskify Demo](https://taskify-nouman.netlify.app/)

## ðŸŒŸ Key Features
- **User Authentication:** Secure login/signup functionality for enhanced privacy.
- **Intuitive Dashboard:** Projects listed on the sidebar with detailed information on the right for easy navigation.
- **Project Management:** Effortlessly add, delete, and update projects to keep everything organized.
- **Task Management:** Manage tasks within projects seamlessly - add, delete, and update with ease.
- **Employee Management:** Efficiently handle employees - add, delete, and update their details effortlessly.
- **Flexible Search:** Swiftly find any information you need using the powerful search feature.
- **Customizable Views:** Tailor the appearance of employee pages to match your preferences.
- **Detailed Task Pages:** Dive deep into task details for comprehensive insights.
- **Personalized Spaces:** Each user enjoys a dedicated account, projects, and tasks for a personalized experience.
- **Dark Mode:** Enhance productivity with a sleek and intuitive dark theme.

## ðŸ”§ Technology Stack
- **Supabase:** Robust database functionality powered by Supabase.
- **React Router Dom:** Smooth navigation within the application for seamless user experience.
- **React Query:** Efficient data fetching, caching, and updates for enhanced performance.
- **Context API:** Effortless management of global state for consistent data handling.
- **Styled Components:** Stylish and polished components created using Styled Components.
- **React Hook Form:** Streamlined form handling for seamless user interactions.
- **React Icons:** Diverse range of icons enhancing the user interface.
- **React Toastify:** User-friendly notifications ensuring smooth interactions.
- **Error Boundaries:** Implemented for enhanced application stability.
- **Date Fns:** Comprehensive date and time functionalities for a seamless user experience.
